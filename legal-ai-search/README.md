# AI Legal Document Search using Endee

Project submitted for Endee.io Internship Evaluation.

## Problem
Lawyers and law students struggle to search long legal documents efficiently.

## Solution
This system uses vector embeddings and the Endee vector database to retrieve relevant legal information and answer questions.

## Features
- Semantic search
- Vector similarity search
- Retrieval Augmented Generation (RAG)
- Fast legal information lookup

## Tech Stack
Python  
Endee Vector Database  
Sentence Transformers  
LLM

## Example Queries
- What is IPC Section 420?
- Punishment for cheating in IPC?
- Explain Article 21.
